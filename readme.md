# _Word Puzzle_

#### _A basic program that uses javascript for loops completed as a project for learnhowtoprogram.com, 10-1-2018_

#### By _**William Kulha**_

## Description

_An application that takes user input in the form of a string, finds vowels in the string and returns the same string with all the vowels replaced with a dash--thus making a word puzzle that can be shared with friends._

## Setup/Installation Requirements

* _Download or clone this repo <code>$git clone https://github.com/zangiboy/word-puzzle.git</code>_
* _Navigate to the folder where you saved this repository._
* _Open index.html with the browser of your choice and enjoy_
